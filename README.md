# What We Lost When The Web Won

Hypermedia and the early internet before the web flattened both. Xanadu, HyperCard, NLS, Gopher, Usenet, Plan 9 — what they were, why they lost, what their loss cost us, and what's being recovered.

**Through-line:** the document lost to the application, and authorship lost with it.

**Read it:** <https://cloudstreet-dev.github.io/What-We-Lost-When-The-Web-Won/>

## Building locally

This is an [mdBook](https://rust-lang.github.io/mdBook/) project.

```sh
cargo install mdbook   # or: brew install mdbook
mdbook serve           # live-reload at http://localhost:3000
mdbook build           # outputs to ./book
```

## Structure

Five parts, twenty-five chapters:

1. **The Frame** — why "won" is the right verb, and what hypermedia and the internet were supposed to be.
2. **The Hypermedia Alternatives** — Xanadu, HyperCard, NLS, Intermedia, Storyspace, NoteCards.
3. **The Internet Alternatives** — Gopher, WAIS, Usenet, email, Plan 9.
4. **How the Web Won** — the technical decisions, the commercial inflection, the standards politics, the platform decade.
5. **What's Recoverable** — dynamic documents, local-first, federation, the static renaissance, personal knowledge tools.
